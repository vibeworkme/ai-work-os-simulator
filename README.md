# Fashion Prompt Studio

패션 모델 스타일을 조합해 Midjourney, DALL-E, Stable Diffusion, Flux용 이미지 생성 프롬프트를 만드는 React 앱입니다.

## 기능

- 22개 패션 스타일 프리셋
- 모델, 헤어, 메이크업, 표정, 촬영 환경, 계절, 소재 옵션
- 상의/하의/아우터/신발/액세서리 의상 빌더
- 랜덤 조합, 초기화, 프롬프트 복사
- 생성기별 프롬프트 형식과 네거티브 프롬프트

## 실행

```bash
npm install
npm run dev
```

## 배포

```bash
npm run build
```
