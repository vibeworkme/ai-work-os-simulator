# AI Learning OS for Knowledge Workers

일반 직장인이 반복 업무를 AI에 맡기기 전에, 업무 기준을 직접 설계해보는 훈련용 웹 앱입니다.

## 왜 이 훈련이 필요한가

AI를 업무에 바로 쓰면 결과가 흔들리기 쉽습니다. 이유는 대개 AI 성능 부족보다 업무 기준이 정리되지 않았기 때문입니다.

- 어떤 업무에 AI를 쓸지 정하지 않았다.
- AI가 참고해야 할 기준 자료가 없다.
- AI에게 맡길 역할이 모호하다.
- 결과물의 출력 형식이 매번 다르다.
- 사람이 무엇을 검토해야 하는지 정해져 있지 않다.

이 훈련은 프롬프트 문장을 멋지게 쓰는 법보다, 업무를 AI가 처리 가능한 형태로 구조화하는 법을 익히는 데 목적이 있습니다.

## 훈련 흐름

1. 업무 선정
   반복되고, 자료가 있고, 사람이 최종 검토할 수 있는 업무를 고릅니다.

2. 기준 자료 구성
   추천 자료를 확인하고, 실제 조직에서 쓰는 최신 자료를 추가합니다.

3. AI 역할 설정
   AI를 일반 도우미가 아니라 특정 업무를 맡는 보조자로 설정합니다.

4. 출력 형식 표준화
   팀원이 같은 기준으로 결과물을 읽고 검토할 수 있도록 항목과 순서를 고정합니다.

5. 검토 루프
   사실, 수치, 사내 기준, 민감정보, 담당자와 기한, 실행 가능성을 사람이 확인합니다.

## 훈련 대상 업무

- 회의록 정리
- 보고서 초안 작성
- 제안서 작성
- 고객 문의 답변
- 품질 이슈 원인 분석
- 생산 현황 요약
- 사내 매뉴얼 검색
- 교육자료 제작

## 벤치마킹 관점

이 앱은 해외 AI 업무 도입 사례에서 공통적으로 나타나는 흐름을 일반 직장인 훈련용으로 단순화했습니다.

- 업무 단위로 AI 적용 범위를 좁힌다.
- 조직의 기준 자료를 함께 사용한다.
- 역할형 AI로 관점을 고정한다.
- 출력 형식을 표준화한다.
- AI 결과물을 초안으로 보고 사람이 검토한다.

## WEAVE& 소개

위브앤은 AI, 창업교육, 진로설계 분야에서 활동하는 교육·컨설팅 기업입니다. 단순히 정보를 제공하는 것을 넘어, 사람들이 자신의 가능성을 발견하고 실제 행동으로 옮길 수 있도록 돕는 도구와 프로그램을 만듭니다.

이 훈련실은 “AI로 사람의 가능성을 발견하고 실행으로 연결한다”는 철학을 직장인의 업무 훈련 맥락으로 확장한 도구입니다.
